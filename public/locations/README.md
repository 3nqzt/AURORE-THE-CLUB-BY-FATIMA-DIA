# Photos des lieux (landing)

Déposez ici les photos utilisées par la section **« Les lieux »** de la page
d'accueil (`src/Landing.jsx`). Les noms de fichiers doivent être **exactement** :

| Fichier         | Lieu sur la page                         |
| --------------- | ---------------------------------------- |
| `rooftop.jpg`   | Toit-terrasse · vue océan                |
| `ocean.jpg`     | Coucher de soleil sur la Corniche        |

- Format conseillé : JPG, largeur ~1200 px, **compressé** (< 300 Ko) pour rester
  rapide en 3G/4G.
- Tant qu'un fichier est absent, un dégradé « coucher de soleil » s'affiche
  automatiquement à la place (aucune image cassée).
- Pour ajouter d'autres lieux, complétez le tableau `LOCATIONS` dans
  `src/Landing.jsx` puis déposez le fichier correspondant ici.
